SECURTHERM: A Hybrid FHE + MPC-FSS Architecture for Battery Monitoring

Author: Rodney Donald HOUNKANRIN Date: May 2026 Framework: SecretFlow (HEU/SPU)
115ms Latency / Sample
98.0% F1 Accuracy
5.4x Speedup vs FHE
8MB Online Bandwidth

Vehicle-to-Grid (V2G) networks enable bidirectional energy exchange between electric vehicles and the power grid, but the associated real-time battery monitoring raises significant privacy concerns: raw thermal data can expose user behavior, location patterns, and driving schedules.

This paper presents SECURTHERM, a novel hybrid architecture that combines Fully Homomorphic Encryption (FHE) with Multi-Party Computation (MPC) using Function Secret Sharing (FSS). By leveraging these technologies, we mitigate the high computational cost of FHE and the communication overhead typical of pure MPC approaches.

The architecture is implemented using the SecretFlow framework. We utilize the Homomorphic Encryption Unit (HEU) for secure linear aggregations and the Secure Processing Unit (SPU) for efficient threshold comparisons via FSS.

Our evaluation on the Kaggle EV Battery dataset (10,000 samples) demonstrates that SECURTHERM matching plaintext performance within statistical tolerance (p = 0.342) while achieving a 5.4× faster inference time and 82% bandwidth reduction compared to traditional MPC-FSS methods.
